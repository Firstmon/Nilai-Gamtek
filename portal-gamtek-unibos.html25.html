⚙️ Konfigurasi Jumlah Gambar per Acara
Atur berapa gambar yang dinilai di setiap acara, lalu klik Simpan Konfigurasi. Semua input nilai akan menyesuaikan.

Upload file yang sudah diisi berdasarkan template. Nilai akan otomatis masuk ke semua mahasiswa sekaligus.
Format kolom: NIM, A1_G1, A1_G2, ... A2_G1, ... A3_G1, ... A4_G1, ..., UTS, UAS, Hadir, TotalHadir
✏️ Input Nilai Manual per Mahasiswa
Pilih mahasiswa untuk input atau edit nilai secara manual.

Export Excel

Rekap PDF

Transkrip Individual
Export transkrip nilai lengkap per mahasiswa ke PDF.

Backup & Restore
Backup semua data ke JSON atau restore untuk sinkronisasi antar perangkat.
